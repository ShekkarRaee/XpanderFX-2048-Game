# XpanderFX
JavaFX 2048 Game.

## How to play?
The UP, DOWN, RIGHT, and LEFT buttons(keyboard) to control tiles in different directions.

The scene's arrow buttons can't be use(i'm coding on it) to control tiles.

## Requirements
Java 8
